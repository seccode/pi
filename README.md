# Pi
The digits of pi are probably not random; we can demonstrate this by predicting the digits with a Random Forest Classifier

# Methodology
The feature vector (x) is the index of the digit of pi. The target vector (y) is (digit in [0,1,2,3,4]).

# Results
From 20,000,000 samples with an expected probability of 0.50015, we got an actual classifier accuracy of 0.5017. The z-score is 13.9 standard deviations.
